# Largest

> Find the largest number in a list


## Install

```
$ npm install --save largest
```


## Usage

```js
var largest = require('largest')

largest(1839, 234, 6346, 3577)
//=> 6346
```

## API

#### `largest(values)` -> `number`

##### values

*Required*
Type: `array[number]`

An array of numbers or a variadic set of numbers.
